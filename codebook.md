# Code Book

## The Subject

An id between 1 and 30 which represent the subject use for the record

## The activity

An label of 6 possibilies us by the subject for the record:

* WALKING
* WALKING_UPSTAIRS
* WALKING_DOWNSTAIRS
* SITTING
* STANDING
* LAYING
